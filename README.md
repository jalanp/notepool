# Notepool

## Getting Started
1. Clone the repository:
```
git clone https://github.com/rkanjani/notepool.git
cd notepool
```
2. Install dependencies:
```
npm install
```
3. Bundle using webpack:
```
webpack
```
4. Start the app:
```
npm run devStart
```
5. App will be running at: http://localhost:3000.
